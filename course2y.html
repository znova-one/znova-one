ကလေးတွေဘာလို့ Programming (Coding) ကိုလေ့လာသင့်သလဲ

Programming (Coding) ဆိုတာဘာလဲ

  • ‘Coding’ ဆိုတဲ့ဝေါဟာရကို အရင်ရှင်းပြရရင်တော့ ကွန်ပျူတာ(Computer) နဲ့ဆက်သွယ်မည့်နည်းလမ်းပါပဲ။
  • ကွန်ပျူတာနဲ့ ဆက်သွယ်ရန်အတွက် ကွန်ပျူတာနားလည်မည့်ဘာသာစကားကို အသုံးပြုရမှာဖြစ်ပြီး တိကျတဲ့လုပ်ငန်းတွေလုပ်ဖို့ ညွန်ကြားချက်တွေပေးရမှာဖြစ်ပါတယ်။
  • ‘Coding’ သင်ယူတာဟာ language သင်တာဖြစ်သလို ညွှန်ကြားချက်တွေပေးတတ်အောင်ပါ တဆက်တည်းသင်ယူသွားမှာပါ။
  • ‘Coding’ တတ်သွားတဲ့အခါ Computer Software, Websites, Applications နဲ့ Video Games တွေကိုဖန်နိုင်သွားမှာဖြစ်ပါတယ်။

Programming သင်ယူတဲ့အခါ ကလေးတွေဘာတွေထပ်ရလာမလဲ

  • Problem Solving - သင်္ချာဘာသာရပ်ရဲ့ သင်္ခန်းစာတွေကိုသုံးပြီး ယုတ္တိရှိရှိနဲ့ ပြဿနာတွေကိုဖြေရှင်းလာနိုင်မှာဖြစ်ပါတယ်။
  • How to Think - မတူညီတဲ့ ရှုထောင့်ပေါင်းစုံကနေ တွေးခေါ်နိုင်စွမ်းတွေရရှိလာမှာဖြစ်ပါတယ်။
  • Creativity - စိတ်ကူးအသိဉာဏ်တွေကို ချဲ့ထွင့်အသုံးပြုတတ်လာမှာဖြစ်ပါတယ်။

ဘယ်အချိန်မှာစလေ့လာသင့်သလဲ

  • နိုင်ငံတကာမှာတော့ ၅ နှစ်အရွယ်လောက်ကစလို့ Coding ရဲ့အခြေခံကျတဲ့ အချက်တွေကို စတင်သင်ကြားနေတာဖြစ်ပြီး ကွန်ပျူတာ အခြေခံနဲ့ ဆက်စပ် အင်္ဂလိပ်စာ ဝေါ်ဟာရတွေကို ဖတ်နိုင်လောက်မည့် အခြေအနေရှိနေလျှင် စတင်သင်ကြားဖို့သင့်လျော်ပါတယ်။

အခု zNova မှာ ကလေးတွေ ကျောင်းတက်ရင်းနဲ့ ကျောင်းစာလဲမထိခိုက်ပဲ ပေါ့ပေါ့ပါးပါး သင်ယူနိုင်မည့် ၂ နှစ်တာ Coding လေ့လာနိုင်မည့် Program တစ်ခုရှိပါတယ်။

၂ နှစ်တာသင်ယူပြီးဘာတွေလုပ်နိုင်မှာလဲ

  • နိုင်ငံတကာက အလုပ်တွေကို အိမ်မှာထိုင်နေရင်းနဲ့ လုပ်ကိုင်နိုင်မည်
  • ဘယ်သွားသွား ဘယ်ရောက်ရောက် Internet နဲ့ Laptop ရှိရုံနဲ့ အလုပ်တွေလုပ်နေနိုင်မယ်
  • သင့်တင့်(မြင့်မား)တဲ့ လစာကြေးငွေနဲ့ Professional တစ်ယောက်အဖြစ်ရပ်တည်နိုင်မည်
  • Freelance လုပ်ကိုင်နိုင်တာကြောင့် လစာအပြင် အပိုဝင်ငွေပါရရှိနိုင်မည်
  • အချိန်ဖြုန်းချင်စိတ်တွေ နည်းပါးလာမည်
  • ကိုယ်တိုင်ဖန်တီးတဲ့အပေါ်မှာ ပြည့်ဝတဲ့ ကျေနပ်ပျော်ရွှင်မှု အမြဲတမ်းရနေမယ်

၂ နှစ်သင်ပြီးဘာတွေထပ်လေ့လာလို့ရနိုင်မလဲ

  • Coding အခြေခံသဘောတရားတွေသိရှိပြီးဖြစ်လို့ တခြားဆက်စပ် ဘာသာရပ်တွေကို ကူပြောင်းလေ့လာနိုင်မှာဖြစ်ပါတယ်။
  • Mobile နည်းပညာတွေနဲ့သက်ဆိုင်တဲ့ Application ရေးသားနည်းများကို ဆက်လေ့လာနိုင်ပါတယ်။
  • Game ရေးသားနည်းနဲ့ဆိုင်တဲ့ အဆင့်မြင်ဆက်စပ်ပညာရပ်တွေကို လေ့လာနိုင်ပါတယ်။
  • Machine Learning (AI) နည်းပညာတွေကို ဆက်လေ့လာနိုင်ပါတယ်။

Programmer တစ်ယောက်အဖြစ်မလုပ်ဖြစ်ခဲ့လျှင့် အချည်းနှီးဖြစ်မှာလား

  • ကလေးတွေဟာ ကိုယ်ပိုင်လွတ်လပ်ခွင့်နဲ့ မိမိရဲ့ Professional ကိုရွေးချယ်ခွင့်ရသင့်ပါတယ်။
  • အင်္ဂျင်နီယာဖြစ်ခွင့်ရှိသလို၊​ ဆရာဝန်ကြီးလည်းဖြစ်နိုင်ပါတယ်။
  • သို့ပေမည့် နည်းပညာခေတ်ဖြစ်သည့်အလျှောက် နည်းပညာနဲ့ကင်းကွာတဲ့ လုပ်ငန်းတွေဟာမရှိတာ့ပါဘူး။
  • ဒါကြောင့် Coding ကိုလေ့လာထားတဲ့သူတစ်ယောက်အနေနဲ့ နည်းပညာသဘောသဘာဝကို သိထားလို့ ဘယ်လမ်းကြောင်းကိုပဲရွေးချယ်ပါစေ အမြဲ နှာတစ်ဖျားအသာနဲ့ လျှောက်လှမ်းနိုင်မှာဖြစ်ပါတယ်။
